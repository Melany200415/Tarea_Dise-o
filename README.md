## Descripción del Proyecto

Sistema de gestión de productos desarrollado en JavaFX que permite el acceso mediante un inicio de sesión y la administración de productos mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

El sistema cuenta con una interfaz gráfica moderna desarrollada con JavaFX y personalizada mediante CSS.

## Funcionalidades Implementadas

### Login

- Validación de usuario y contraseña.
- Selección de rol.
- Acceso al dashboard.

### Dashboard

- Menú lateral de navegación.
- Acceso al módulo de productos.
- Opción de cerrar sesión.

### Gestión de Productos

- Registrar productos.
- Mostrar productos en tabla.
- Actualizar productos.
- Eliminar productos.
- Limpiar formulario.
- Validación de campos.
